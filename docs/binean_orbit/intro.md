# Orbit — The Core of Modernization

> **Transform your insurance core into a modern, API-driven platform.**

---

## What Orbit Delivers

Orbit is the next phase of the Nova transformation, designed to modernize the heart of your insurance operations:

- **REST API Layer** — Transform Ingenium into a RESTful service, enabling direct integration with modern applications without MQ middleware
- **Middleware Elimination** — Remove dependency on IBM MQ/ActiveMQ, reducing complexity and licensing costs
- **Performance Migration** — Begin migrating critical business functions from COBOL to high-performance Rust
- **24/7 Availability** — Query policy information even during batch processing windows

---

## Business Impact

| Current State | After Orbit |
|--------------|-------------|
| MQ middleware required for every integration | Direct REST API calls |
| Batch windows block online access | Always-on availability |
| COBOL-only business logic | Rust services for critical paths |
| Complex middleware licensing | Simplified, lower-cost architecture |
