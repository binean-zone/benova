# The Nova Platform

> A practical modernization platform for insurers running Ingenium.

![Nova Architecture](/img/nova.png)

## Executive summary

Nova modernizes the Ingenium life-insurance core in phases, without a big-bang replacement.

- Proven in production on a real programme
- AIX-to-Linux migration completed in around 9 months
- Representative deployment time reduced from around 3 hours to around 20 minutes
- Planned downtime window around 5 minutes per deployment

## What Nova changes first

1. Standardize build and deployment with the Nexus toolchain in VS Code
2. Reduce manual environment drift and release risk
3. Open selected core capabilities via REST APIs
4. Preserve existing Ingenium logic and business ownership

## Delivery model

- POC-first, then phased scale-out
- Go/no-go gates at each phase
- Cloud, on-premise, or hybrid
- Commercial model supports fixed-scope milestones or time-and-material
