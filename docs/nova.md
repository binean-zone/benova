## Platform Architecture

Nova is organized into four clear layers, each modernized independently:

### Presentation Layer
**Modern, responsive interfaces** built with React — delivering the digital experience your customers and agents expect. Supports REST and GraphQL for flexible data access.

### Integration Layer
**Open connectivity** through API Gateway and Event Bus — connect to any external system, partner, or digital channel. *Optional — deploy only if your infrastructure needs it.*

### Application Layer
**High-performance core services** built in Rust — the same speed as hand-optimized C with complete memory safety. Microservices design means each component scales independently.

### Data Layer
**Flexible, future-proof data** — maintain your existing Db2 for stability while expanding to modern distributed databases (YugabyteDB) for cloud-native scalability.
