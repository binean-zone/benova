# Ingenium Modernization for Insurance Core Systems

Insurers running Ingenium need a modernization path that improves delivery speed without a risky core replacement.

Nova provides a phased approach designed for insurance core modernization with clear decision gates.

## What decision makers are searching for

- How to modernize Ingenium without a big-bang migration
- How to reduce deployment time for COBOL-based insurance core changes
- How to move from AIX to Linux with controlled risk
- How to expose insurance core capabilities via APIs

## Production-grounded outcomes

- Proven in production
- Around 9 months for AIX-to-Linux migration in the reference programme
- Deployment time reduced from around 3 hours to around 20 minutes for a representative change
- Planned downtime window around 5 minutes per deployment

## Core modernization scope

1. Foundation: AIX to Linux migration
2. Nexus: automated COBOL build and deployment workflows
3. Orbit: API-led integration for core capabilities
4. Governance: phased go/no-go decisions based on KPI evidence
